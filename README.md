ASSETS CLEANER
==============

ENGLISH
-------
Assets Cleaner is a browser-based tool designed to remove unused assets from Spine project folders.

The tool analyzes a .atlas file and compares it with a selected assets directory. It identifies which
assets are required and which are unused, then allows the user to download a cleaned archive that
contains only the necessary files while preserving the original folder structure.

Key features:
- Runs completely in the browser
- No files are uploaded to any server
- Preserves original folder structure
- Supports PNG, JPG, JPEG files
- Works with nested folders
- Handles spaces, special characters, and Unicode filenames

How to use:
1. Upload a .atlas file
2. Select the root folder containing the assets
3. Click "Analyze"
4. Review required and unused assets
5. Click "Clean & Download"

УКРАЇНСЬКА
----------
Assets Cleaner — це браузерний інструмент для видалення невикористаних асетів із папок проектів Spine.

Інструмент аналізує файл .atlas та порівнює його з вибраною папкою з асетами. Він визначає,
які файли використовуються, а які є зайвими, після чого дозволяє завантажити очищений архів,
що містить тільки потрібні файли із збереженням оригінальної структури папок.

Основні можливості:
- Працює повністю у браузері
- Файли не завантажуються на сервер
- Зберігає структуру папок
- Підтримує PNG, JPG, JPEG
- Працює з вкладеними папками
- Підтримує назви файлів з пробілами, спецсимволами та Unicode

Як користуватися:
1. Завантажте файл .atlas
2. Виберіть кореневу папку з асетами
3. Натисніть "Аналіз"
4. Перегляньте потрібні та зайві файли
5. Натисніть "Очистити і завантажити"
